# nem2(catapult) development Utility Interface

This is a tools for nem2.

## 🐗 DEMO

* [nem2 Dev Utility Interface](https://nem2-dev-ui.netlify.com/)

## 🔧 Features

* Account Generation
* Fetch Account Info
* Draw MLMS Structure in text
* Fetch Namespace Info
* Fetch Mosaic Info
* Distribute XEM for Addresses
* Block Monitoring
* Some Useful Tools

## 🔨 Build

```
# install packages
$ yarn

# build
$ yarn build

# development
$ yarn start
```

Good luck!
